## Project Overview

Personal portfolio/website for Jeezman (Tobi Adeyemi). Static site deployed to GitHub Pages at `jeezman.github.io`.

## Commands

- `pnpm dev` — start Vite dev server
- `pnpm build` — production build (outputs to `dist/`)
- `pnpm preview` — preview production build
- `pnpm lint` — ESLint
- `pnpm deploy` — deploy to GitHub Pages via `gh-pages -d dist`

## Architecture

Vite + React 19 + TanStack Router with static build. Uses Tailwind CSS 3 with `@tailwindcss/typography`, and `next-themes` for dark mode.

**Production paths**: In production, Vite `base` and TanStack Router `basepath` are set to `/jeezman` (for GitHub Pages subdirectory hosting).

**Key structure**:
- `src/routes/` — TanStack Router file-based routes. `__root.tsx` is the root layout, `index.tsx` is the homepage.
- `src/components/` — Shared React components (Header, Footer, Layout, Container, icons).
- `src/provider.tsx` — Client-side Providers wrapping ThemeProvider + AppContext (tracks previous pathname).
- `src/constants.ts` — Social link URLs (GitHub, X, Nostr).
- `src/main.tsx` — App entry point, creates router and renders to DOM.
- `src/globals.css` — Tailwind imports + @font-face declarations for Geist fonts.
- `typography.ts` — Custom Tailwind typography plugin styles with full light/dark mode prose token system.
- `public/fonts/` — Geist Sans and Geist Mono font files.

**Path alias**: `@/*` maps to the project root.

**Dark mode**: Uses Tailwind `selector` strategy, controlled by `next-themes` with `attribute="class"`.

**Routing**: TanStack Router with file-based routing. Route tree is auto-generated by `@tanstack/router-plugin/vite`.
